## GRPC-NOTES
### Installation gRPC Protobuf
[Installation](https://grpc.io/docs/protoc-installation/#binary-install)
```vim
//MacOS
$ brew install protobuf
$ protoc --version 
```
```vim
//Linux
$ apt install -y protobuf-compiler
$ protoc --version 
```
